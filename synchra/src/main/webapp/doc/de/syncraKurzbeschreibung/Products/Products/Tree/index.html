Im Register Baum wird die Baumstruktur des Produktes beginnend mit der Baugruppe, die bei der Anlage des Produktes entstanden ist, angezeigt. Über die Baugruppenstruktur kann man sich bis auf die Einzelteilebene herunterhangeln.

Ziel, Plan und Prognosepreis werden dabei für jede Zeile einzeln in Abhängigkeit des Status der Maßnahmen ausgerechnet. Dabei muß darauf geachtet werden, dass der Spaltenwert eines übergeordneten Knotens nicht zwingend die Summe der darunterliegenden Knoten ist. Wenn eine Maßnahme für eine Komponente definiert wird, die kein Einzelteil ist, also eine Baugruppe) so wird die Auswirkung dieser Maßnahme auf diese Komponente direkt auf der Komponente gebucht. Diese Auswirkung wird in den Spalten "Eigenanteil Prognose" oder "Eigenanteil Plan" ausgewiesen.
