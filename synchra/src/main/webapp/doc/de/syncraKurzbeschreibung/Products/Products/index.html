Im Register Produkte muss zunächst aus der Menge der definierten Produkte eines ausgewählt werden. Wird dazu ein Referenzdatum angegeben, so beziehen sich die Auswertungen (sofern nicht selber datumsabhängig) auf dieses Referenzdatum. Ein Klick auf den Aktualisierungsbutton baut nun den entsprechenden Produktbaum auf.
