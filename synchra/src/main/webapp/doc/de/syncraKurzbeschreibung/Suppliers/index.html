• Zuordnung von Gesellschaften als Lieferanten von Komponenten
  • Definition von Ansprechpartnern innerhalb der Gesellschaft

Das Register Lieferanten macht an mehreren Stellen Gebrauch von den Werkzeugen der Modellierung:

1. wird der Ansprechaprtner als Untertyp von Person modelliert und hat somit keine eigenen Attribute

2. wird die Beziehung zwischen dem Objekttyp Gesellschaft und dem Objekttyp Ansprechpartner als Komposition modelliert (ohne die Gesellschaft wäre der Ansprechpartner eine Datenleiche), wodurch sich im Rahmen der Oberflächenmodellierung die Möglichkeit ergibt, einen Ansprechpartner im Kontext der Gesellschaft ohne weitere Formulardefinition anzulegen.
